Flight In-situ Data Acquisition

Integrating environmental sensing and real-time telemetry into UAV platforms.

Research Methodology

  • Integrated environmental sensors with an Arduino-based Data Acquisition System (DAS).
  • Implemented telemetry modules for wireless communication between UAV and ground station.
  • Mounted the DAS on a quadcopter and conducted multiple test flights.
  • Live sensor data transmitted to the ground station in real-time via telemetry link.
  • Processed and visualized captured data into 2D/3D graphs using Python scripts.

Conclusion

  • Captured live environmental data (temperature, humidity) during flight.
  • Enabled real-time decision support through telemetry + visualization.
  • Demonstrated future potential: pollution source detection and wildlife monitoring.
  • DAS integration is a key enabler for autonomous, data-driven missions.
